# Deep Learning Repository

Welcome to my **Deep Learning** repository 🚀.  
This repository is a continuously growing collection of deep learning concepts, implementations, experiments, and results using Python and modern frameworks like TensorFlow/Keras and PyTorch.

The goal is to build a structured learning resource that covers:

- Fundamental neural networks  
- Convolutional Neural Networks (CNN)  
- Computer Vision tasks  
- Optimization techniques  
- Model evaluation and analysis  
- Real-world deep learning projects

---

## 📌 What This Repository Contains

Currently the repository includes:

### 🔹 Image Classification Projects
- CNN on **Fashion MNIST**  
- CNN on **CIFAR-10**

### 🔹 Model Evaluation
- Accuracy & Loss visualization  
- Confusion Matrix  
- Classification Report  
- Performance comparison between datasets

### 🔹 End-to-End Workflow
- Data preprocessing  
- Model design  
- Training & validation  
- Result interpretation
